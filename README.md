# Backup de tesis: retinopatía diabética

Este repositorio contiene una copia liviana de trabajo para la tesis sobre clasificación de retinopatía diabética.

Incluye:
- notebooks de entrenamiento, validación, matrices y tablas;
- CSV de índices, etiquetas, métricas y reportes;
- configuraciones de experimentos;
- tablas finales y archivos de resumen.

No incluye, por tamaño y por límites de GitHub:
- imágenes crudas de fondo de ojo;
- checkpoints `.pth`;
- modelos serializados `.pkl`;
- caches `.npz` / `.npy`;
- archivos locales de Weights & Biases;
- gráficos/imágenes generadas.

Los datos pesados deben respaldarse por separado en almacenamiento externo, Drive, OneDrive o Git LFS con cuota suficiente.
